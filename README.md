# Stack Quiz

Интерактивный тренажёр для подготовки к собеседованиям по веб-технологиям. Проверяй знания в дороге, в очереди или в любое свободное время. Включает квизы по теории и полноценный онлайн редактор кода с алгоритмическими задачами.

**Ссылка на приложение:** https://gevorg22.github.io/stack-quiz/

## Категории вопросов

**JavaScript** вопросы по замыканиям, промисам, Event Loop, прототипам, ES2022+, JIT-компиляции, бандлерам, паттернам проектирования и оптимизации.

**React** вопросы по хукам, Concurrent Mode, Server Components, React 19, архитектурным паттернам, производительности и экосистеме.

**TypeScript** вопросы по системе типов, дженерикам, утилитарным типам, декораторам и продвинутым паттернам.

**HTML** вопросы по семантике, доступности, Web API, PWA, производительности и современным браузерным возможностям.

**CSS** вопросы по специфичности, Flexbox, Grid, Container Queries, Cascade Layers, анимациям и современным CSS-функциям.

**Сети и Web** вопросы по HTTP/2/3, CORS, REST, WebSocket, безопасности, HTTPS, кэшированию и браузерным API.

**Git** вопросы по командам, ветвлению, rebase, внутреннему устройству, хукам и стратегиям работы в команде.

**Все темы** все вопросы по всем технологиям вперемешку для комплексной подготовки.

## Возможности

**Уровень сложности** выбирай Базовый, Средний или Продвинутый (или Любая) перед стартом. Каждый вопрос помечен цветным бейджем прямо во время квиза.

**Выбор количества вопросов** выбирай 10, 25, 50 или все вопросы перед стартом. Доступное количество обновляется в зависимости от выбранной сложности.

**Тёмная тема** переключается кнопкой, сохраняется между сессиями.

**Закладки** сохраняй интересные вопросы и проходи квиз только из закладок.

**История** результаты всех квизов хранятся локально, можно очистить.

**Повторить ошибки** после квиза можно пройти заново только те вопросы, где ошибся.

**Серия ответов (streak)** счётчик правильных ответов подряд с анимацией огня.

**Клавиатурные сокращения** 1/2/3/4 или A/B/C/D для выбора ответа, Enter для следующего вопроса, Escape для закрытия пояснения.

**Конфетти** при результате 100% запускается анимация.

**PWA** можно добавить на домашний экран смартфона как отдельное приложение.

**Адаптивная вёрстка** работает на телефоне, планшете и десктопе.

## Редактор кода

Полноценный онлайн редактор в стиле технических собеседований Яндекса со встроенной консолью и автоматической проверкой решений.

**80+ задач** разделённых по уровням сложности: Начальный, Средний, Продвинутый.

**Алгоритмические задачи** сортировки (пузырьком, слиянием, быстрая), скользящее окно, два указателя, бинарный поиск, динамическое программирование, структуры данных.

**Задачи на структуры данных** стек с минимумом, очередь через два стека, разворот связного списка, спиральная матрица, количество островов (BFS/DFS).

**Задачи на строки** анаграммы, группировка анаграмм, самая длинная подстрока без повторений, сжатие строки, палиндромная подстрока, римские цифры.

**Задачи по JavaScript** реализация debounce, throttle, memoize, curry, EventEmitter, Promise.all, глубокое клонирование и сравнение объектов, реализация Array.map и Array.reduce.

**Автопроверка** каждая задача содержит тест-кейсы, которые выполняются в изолированной sandbox-среде через iframe.

**Подсказки** у каждой задачи есть кнопка с подсказкой по алгоритму решения.

**Решено** отметки о решённых задачах сохраняются в localStorage.

**Playground** свободный редактор без задания для экспериментов с любым JS кодом.

**Мобильная версия** на телефоне задание и редактор переключаются вкладками.

**Растягиваемая консоль** высоту консоли можно менять перетаскиванием за ручку мышью или пальцем.

## Стек технологий

**Next.js 16** с App Router и статической генерацией.

**React 19** с TypeScript.

**Tailwind CSS v4** для адаптивной вёрстки и тёмной темы.

**Monaco Editor** редактор кода (тот же что используется в VS Code).

**Vitest** с Testing Library для юнит- и компонентных тестов (118 тестов).

**ESLint** с eslint-config-next и eslint-config-prettier.

**Prettier** для форматирования кода.

**GitHub Actions** для автоматического деплоя на GitHub Pages.

## Архитектура

```
src/
  app/           Next.js App Router (layout, page, manifest)
  components/    UI компоненты
    QuizClient       главный контроллер навигации
    QuizRunner       изолированная логика одной сессии квиза
    QuizQuestion     экран вопроса с клавиатурными сокращениями
    QuizSetup        выбор сложности и количества вопросов
    ResultScreen     итоги с историей и кнопкой повторения ошибок
    ExplanationModal модалка с пояснением и закладкой
    AnswerOption     вариант ответа с подсветкой
    ProgressBar      прогресс и streak
    CategoryCard     карточка категории
    BookmarksView    список закладок
    BookmarkButton   кнопка закладки
    StreakBadge      бейдж серии правильных ответов
    DarkModeToggle   переключатель темы
    Confetti         анимация при 100%
    ChallengesView   список задач с фильтрацией и прогрессом
    ChallengeRunner  IDE с описанием задачи, редактором и консолью
    CodeEditor       обёртка Monaco Editor с поддержкой тем
    Playground       свободный редактор без задания
  data/
    questions/     базы вопросов (javascript, react, typescript, html, css, network, git)
    challenges.ts  80+ задач для редактора кода
  hooks/
    useQuiz          логика квиза (streak, wrongQuestions)
    useCodeRunner    запуск кода в изолированном iframe sandbox
    useLocalStorage  универсальный хук для localStorage
    useDarkMode      тёмная тема
    useBookmarks     закладки
    useHistory       история прохождений
  types/           TypeScript типы (quiz, challenge)
  utils/           shuffle (алгоритм Фишера-Йейтса)
  test/            конфигурация тестового окружения
```

## Запуск локально

```bash
npm install
npm run dev
```

Открой http://localhost:3000

## Тесты

```bash
npm test
```

118 тестов: утилиты, валидация банка вопросов (включая difficulty), хуки, компоненты.

## Сборка

```bash
npm run build
```

Статические файлы генерируются в папку `out/`.

## Деплой

Деплой происходит автоматически через GitHub Actions при пуше в ветку `main`. Приложение публикуется на GitHub Pages.
